# Predictive Analytics for Vehicle Maintenance in Fleet Management

## Project Overview

This project uses Predictive Analytics and Machine Learning to identify vehicles that are likely to require maintenance before breakdowns occur.

The objective is to improve fleet reliability, reduce downtime, and optimize maintenance scheduling through data-driven decision-making.

---

## Business Problem

Unexpected vehicle failures can lead to:

- Increased maintenance costs
- Vehicle downtime
- Operational delays
- Reduced fleet efficiency

---

## Project Objectives

- Analyze vehicle maintenance data
- Identify factors influencing maintenance requirements
- Build predictive models for maintenance prediction
- Generate actionable business insights
- Develop an interactive Power BI dashboard

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- SQL
- Power BI
- Excel

---

## Machine Learning Models Used

1. Logistic Regression
2. Decision Tree
3. Random Forest

---

## Key Findings

- Vehicle Age significantly impacts maintenance requirements.
- Higher mileage vehicles are more likely to require maintenance.
- Brake Condition, Battery Status, and Reported Issues are strong predictors.
- Random Forest achieved the highest model performance.

---

## Dataset Features

- Vehicle Age
- Mileage
- Fuel Type
- Engine Size
- Brake Condition
- Tire Condition
- Battery Status
- Service History
- Maintenance History
- Reported Issues
